# Architecture Decision Records (ADR)

Este diretório contém todas as decisões arquiteturais oficiais do projeto Circulou Backend.

As ADRs representam decisões permanentes da arquitetura do sistema e devem ser consideradas durante qualquer evolução do projeto.

Toda implementação futura deverá respeitar as ADRs aqui documentadas.

Novas ADRs deverão ser aprovadas antes da implementação de alterações arquiteturais relevantes.
